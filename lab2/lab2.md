# Внешнее окружение и границы системы AeroProduction ERP

| Стейкхолдер | Интерес к системе | Цели | Влияние на проект | Риски |
|-------------|-------------------|------|-------------------|-------|
| **Генеральный директор** | Контроль производства и стратегические решения | Повышение эффективности производства, снижение издержек | Высокое (утверждает бюджет, принимает решение о внедрении) | Отказ от проекта при неудовлетворительных результатах |
| **Начальники цехов** | Оперативное управление цехом | Контроль выполнения планов, управление ресурсами | Среднее (участвуют в определении требований) | Сопротивление изменениям, саботаж внедрения |
| **Начальники участков** | Управление участком и бригадами | Распределение работ, контроль сроков | Среднее (основные пользователи) | Недостаточная квалификация для работы с системой |
| **Мастера** | Назначение бригад на работы | Оперативный контроль выполнения задач | Низкое (пользователи) | Игнорирование системы, ведение параллельного учета |
| **Отдел кадров** | Учет персонала | Автоматизация кадрового учета, отчетность | Среднее (ответственны за данные о персонале) | Несвоевременное обновление данных |
| **ПЭО (планово-экономический отдел)** | Планирование производства | Оптимизация загрузки, планирование выпуска | Высокое (определяют производственные процессы) | Конфликт с существующими методиками планирования |
| **Служба качества** | Контроль испытаний | Отслеживание прохождения изделиями испытаний | Среднее (пользователи модуля испытаний) | Недостаточная детализация данных об испытаниях |
| **Испытатели** | Регистрация результатов испытаний | Упрощение документирования испытаний | Низкое (пользователи) | Ошибки при вводе данных |
| **Бухгалтерия** | Учет выпуска, расчет зарплаты | Получение данных для финансового учета | Среднее (требуют интеграции с 1С) | Несоответствие форматов данных |
| **IT-отдел** | Эксплуатация системы | Стабильная работа, минимум инцидентов | Высокое (отвечают за внедрение и поддержку) | Нехватка компетенций, перегрузка |
| **Рабочие и бригадиры** | Косвенное использование | Прозрачность распределения работ | Низкое (не являются прямыми пользователями) | Недовольство изменением процессов |
| **Поставщики оборудования** | Данные об оборудовании | Интеграция данных об оборудовании | Низкое | Отсутствие стандартизированных данных |

### Что ВХОДИТ в систему:

**Модуль организационной структуры:**
- Управление цехами и участками
- Назначение руководителей

**Модуль управления персоналом:**
- Учет ИТР и рабочих
- Управление бригадами
- Учет движения кадров (прием, увольнение, перемещение)
- Хранение истории изменений

**Модуль управления изделиями:**
- Справочник категорий и видов изделий
- Регистрация изделий со специфическими атрибутами
- Отслеживание статуса изделий

**Модуль производственных процессов:**
- Определение циклов работ
- Назначение бригад на участки
- Отслеживание перемещения изделий
- Контроль сроков выполнения работ

**Модуль испытаний:**
- Управление лабораториями и оборудованием
- Регистрация испытаний
- Фиксация результатов испытаний
- Учет задействованного оборудования и испытателей

**Модуль отчетности:**
- Все 14 видов запросов из требований
- Дашборды для разных ролей
- Экспорт отчетов

**Модуль администрирования:**
- Управление пользователями и ролями
- Журнал аудита
- Настройка системы

### Что НЕ ВХОДИТ в систему:

- Бухгалтерский учет (остается в 1С)
- Расчет заработной платы
- Управление финансами и бюджетированием
- Конструкторская документация (САПР)
- Технологическая подготовка производства (ТПП)
- Управление закупками и снабжением
- Складской учет материалов и комплектующих
- Управление договорами и контрактами
- Автоматизация работы станков и оборудования (АСУ ТП)
- Документооборот (СЭД)
- Система контроля доступа на территорию
- Видеонаблюдение

### Внешние системы, взаимодействующие с AeroProduction ERP:

| Внешняя система | Тип интеграции | Направление данных |
|-----------------|----------------|-------------------|
| **1С:Бухгалтерия** | REST API / выгрузка файлов | Экспорт данных о выпущенных изделиях, персонале |
| **1С:Зарплата и управление персоналом** | REST API / выгрузка файлов | Экспорт данных о персонале, отработанном времени |
| **Система электронного документооборота (СЭД)** | REST API | Импорт приказов о приеме/увольнении, экспорт отчетов |
| **САПР-система (КОМПАС, AutoCAD)** | Импорт файлов | Импорт конструкторской информации об изделиях |
| **Система управления качеством** | REST API | Экспорт данных об испытаниях |
| **Active Directory / LDAP** | LDAP | Аутентификация пользователей |
| **Email-сервер** | SMTP | Отправка уведомлений |

### Процессы, подлежащие автоматизации:

**Подлежат автоматизации:**
1. Регистрация изделий в производстве
2. Назначение бригад на выполнение работ
3. Отслеживание перемещения изделий между участками
4. Фиксация начала/окончания работ на участках
5. Регистрация испытаний и их результатов
6. Учет движения кадров
7. Формирование всех видов отчетов
8. Контроль загрузки участков и бригад
9. Учет использования оборудования лабораторий
10. Формирование статистики выпуска

**Частично автоматизируются:**
1. Планирование производства (система предоставляет данные, решение принимает ПЭО)
2. Формирование бригад (рекомендации на основе загрузки, решение принимает начальник участка)
3. Назначение руководителей (фиксация в системе, решение принимает руководство)

**НЕ подлежат автоматизации (остаются ручными):**
1. Принятие стратегических решений
2. Непосредственное выполнение работ по сборке
3. Проведение физических испытаний
4. Собеседования при приеме на работу
